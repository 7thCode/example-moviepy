# example-moviepy

タイトルを合成したクリップと他２つのクリップをクロスフェードで繋いで、BGMを付けて最初と際顔をフェード。

# リソースファイル構成

```
Desktop
    |
    |- video
         |
         |- clip0.mp4   // title
         |
         |- clip1.mp4   // content
         |
         |- clip2.mp4   // ending
         |
         |- bgm.m4a     // bgm 
         |
         |- ipag.ttf    // font
```
